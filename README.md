# Digital Nurture 5.0 - Java FSE Upskilling

A Java Full Stack development practice repository created as part of the Digital Nurture 5.0 Java FSE Upskilling program.  
It contains structured implementations covering backend development, database concepts, and frontend technologies.

## Modules

### Core Java

Hands-on Java programming covering:

- Java fundamentals
- Object-Oriented Programming (OOP)
- Collections Framework
- Exception Handling
- File Handling
- Multithreading
- Streams and Lambda Expressions
- JDBC
- Modern Java features

Repository:
`core-java-practice`

---

### ANSI SQL using MySQL

Database practice covering:

- SQL fundamentals
- Data retrieval queries
- Filtering and sorting
- Aggregate functions
- Joins and subqueries
- Database modifications
- Constraints and indexing
- HackerRank and LeetCode SQL practice

Repository:
`ANSI-SQL-practice`

---

### Frontend Development

Frontend fundamentals covering:

- UI components and utilities
