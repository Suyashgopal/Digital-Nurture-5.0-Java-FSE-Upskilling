# Running the Code

## Clone Repository

```bash
git clone --recursive https://github.com/Suyashgopal/Digital-Nurture-5.0-Java-FSE-Upskilling.git
```

Navigate into the repository:

```bash
cd Digital-Nurture-5.0-Java-FSE-Upskilling
```

---

## Run Java Programs

Navigate:

```bash
cd core-java-pratice
```

Compile:

```bash
javac FileName.java
```

Execute:

```bash
java FileName
```

---

## Run SQL Files

Open MySQL:

```bash
mysql -u username -p
```

Execute:

```sql
SOURCE filename.sql;
```
