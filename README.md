## Running Java Programs

Clone the repository:

```bash
git clone https://github.com/Suyashgopal/Digital-Nurture-5.0-Java-FSE-Upskilling.git
```

Navigate:

```bash
cd core-java-pratice
```

Compile:

```bash
javac FileName.java
```

Run:

```bash
java FileName
```

Example:

```bash
javac OOPDemo.java

java OOPDemo
```
